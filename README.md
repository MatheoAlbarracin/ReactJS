# En este proyecto comence de la siguiente manera:

Comence primero pensando que iba a tener de idea principal, en la que se me ocurrio, una pagina de cosas para la PC, monitores, teclados, componentes, etc; En la que se llamaba "Gaming4u", Para eso cree lo que viene siendo mi NavBar primero donde se enfocaba en 3 tipos: "Componentes, Monitores y Agregados".
Luego de crear el NavBar continue por los items, cargandolos a FireBase, una pagina de gran ayuda y facilidad a la hora de cargar productos. 
En la siguiente idea fui por lo mas dificil que fue los Checkouts y por ultimo fui por el carrito de compras donde suba todo lo que es la data de los items.
Por suerte para este proyecto pudimos crear un grupo de programadores que cursabamos y nos pudimos ayudar mucho mutuamente, tanto mi ayuda como la que recibi, y los tutores CoderAsk y el profesor.
